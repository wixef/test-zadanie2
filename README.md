# MONO — каталог кофе

Практическое задание 2: секция каталога интернет-магазина на чистом HTML + CSS.

## Что сделано

- 8 карточек товара в адаптивной сетке (1 → 2 → 3 → 4 колонки)
- Заголовок секции, описание, фильтры-пилюли
- Hover-состояния: подъём карточки, zoom изображения, заливка кнопок
- Типографика: Instrument Serif + DM Sans
- Семантическая разметка: `main`, `article`, `figure`, `nav`

## Запуск локально

```bash
open index.html
# или
python3 -m http.server 8080
```

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы
3. Settings → Pages → Source: Deploy from branch → `main` / `/ (root)`
4. Ссылка будет: `https://<username>.github.io/<repo>/`
